
Start Command:
* This command need to start game.
* Based example to use: <start> [SMTH]
BASED:
    * start [NOTHING] - printing this message

START LEVEL:
    * start [l] - starting new level and open the game
        NOTE: You will be started level that was been created automatically

START CUSTOM LEVEL:
    * start [LEVEL NAME] - starting already created level and open the game
        NOTE: This using when you want to start already created level.
        !!! For use this modified command you must think name of level that you want to open. !!!
