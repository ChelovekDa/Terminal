
Use Command:
* Эта команда нужна для использования некоторых предметов внутри уровня или на нем.
* Базовый пример использования: <use> [ЧТО-ТО] on [ЧТО-ТО]
ИСПОЛЬЗОВАНИЕ ПО УМОЛЧАНИЮ:
    * use [НИЧЕГО] - выводит это сообщение

USE KEY:
    * use [ИМЯ КОМНАТЫ] on [КООРДИНАТЫ] - использование ключа из указанной комнаты 
    на заблокированной двери комнаты по координатам
        Пример использования: use A3 on 0.2.A5
        A3 - имя комнаты, в которой хранится ключ (потому что все имена комнат уникальные).
        0.2.A5 - координаты комнаты, на которой вы хотите использовать ключ. 
        (0 - номер здания, 2 - номер этажа, A5 - имя комнаты)

        NOTE: Вы должны вводить имя комнаты с включенным капс локом!

USE DOCUMENT:
    * use [doc] in [КООРДИНАТЫ] [МОДИФИКАТОР] - использование документа
        doc - этот аргумент нужен для понимания того, что вы хотите сделать, 
        использовать или прочитать этот документ.
        [КООРДИНАТЫ] - аргумент с координатами документа.
        [МОДИФИКАТОР] - аргумент для понимания того, что вы хотите сделать - [read/fin]

        Пример использования: use doc in 0.2.A5 finish
        doc - нужный аргумент, он всегда такой
        0.2.A5 - координаты комнаты, в которой вы хотите использовать документ
        finish - этот модификатор определяет что вы хотите сделать

        NOTE: Вы должны вводить имя комнаты с включенным капс локом!

        Вы можете указать эти модификаторы:
            * для чтения документа: [r], [read], [re]
            * для завершения уровня: [fin], [finish], [f]

        NOTE: Другие модификаторы не могут быть введены!
