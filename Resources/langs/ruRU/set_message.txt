
Set Command:
* Эта команда нужна для установки некоторых значений внутри Виртуального Окружения (VE) или в Терминале.
* Базовый пример использования: <set> [ЧТО-ТО]
ИСПОЛЬЗОВАНИЕ ПО УМОЛЧАНИЮ:
    * set [НИЧЕГО] - выводит это сообщение.

SET LANGUAGE:
    * set [lang] [ЧТО-ТО] - устанавливает введенный язык.
    NOTE: вместо второго аргумента может быть указано следующее: [lan], [lang], [language]
    Вы должны вводить вместо <[ЧТО-ТО]> некоторый язык.
    По умолчанию могут быть введены: английский или русский языки.
