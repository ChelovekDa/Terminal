
Virtual Environment Commands (VEC):
* help - выводит это сообщение.
* create - создает что-либо в Виртуальном Окружении (VE).
* start - команда для начала игры.
* c/clear - команда для прокрутки консоли Терминала вниз.
* del/delete - полностью очищает консоль Терминала.
* set - команда для установки значений.

Для полной информации напишите название команды.

NOTE: Вы можете использовать эти команды даже в игре, для этого напишите <ve (ВАША КОМАНДА)>
Но для некоторых изменений может потребоватся перезапуск игры.
NOTE: Вы можете копировать последний введенный текст в консоль Терминала.
Для этого, нажмите 'shift+insert'
