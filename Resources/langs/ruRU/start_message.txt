
Start Command:
* Эта команда нужна, чтобы начать игру.
* Базовый пример использования: <start> [ЧТО-ТО]
ИСПОЛЬЗОВАНИЕ ПО УМОЛЧАНИЮ:
    * start [NOTHING] - выводит это сообщение

START LEVEL:
    * start [l] - запускает новый уровень и открывает игру
        NOTE: Таким образом вы запускаете уровень, созданный автоматически

START CUSTOM LEVEL:
    * start [LEVEL NAME] - запускает уже созданный уровень и открывает игру
        NOTE: Этот вариант команды используется только тогда, когда вы хотите запустить уже созданный уровень.
        !!! Для использования этого варианта команды вы должны знать имя уровня, который вы хотите открыть. !!!
