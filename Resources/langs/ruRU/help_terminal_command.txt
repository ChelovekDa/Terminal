
The Terminal Commands (TTK):
* help - выводит это сообщение.
* VE/ve - нужно для использования команд виртуального окружения (VE)
    NOTE: после специальных символов должна быть команда, которую вы хотите использовать.
* ping - выводит список со всеми комнатами, содержащими переданный предмет.
* use - нужна для использования некоторых предметов внутри игры.
* c/clear - листает консоль вниз.
* list - выводит информацию о объекте.
* del/delete - полностью очищает консоль Терминала.

Для получения большей информации напишите имя команды в консоль.

NOTE: Вы можете копировать последний введенный вами текст в консоль Терминала.
Для этого нажмите сочетание 'shift+insert'
