
Create Command:
* Эта команда создает некоторые вещи в Виртуальном Окружении (VE), не принадлежащие к основному Терминалу.
* Базовый пример использования: <create> [ЧТО-ТО]
ИСПОЛЬЗОВАНИЕ ПО УМОЛЧАНИЮ:
    * create [НИЧЕГО] - выводит это сообщение

CREATE LEVEL:
    * create [l] - создает новый уровень для дальнейшего использования.
    * create [new_level (ИЛИ ДРУГОЕ ИМЯ)] - создает новый уровень для дальнейшего использования,
    но с определенным названием.
        NOTE: Имя уровня не может содержать пробелы.
    * create [new_level (ИЛИ ДРУГОЕ ИМЯ)] [1/2/3 - СЛОЖНОСТЬ]
        NOTE: Сложность создавалась с логикой 'чем выше - тем сложнее',
        поэтому 1 - легко, 2 - средне (по умолчанию), 3 - сложно (не рекомендуется для первой игры)
        !!! Вы не можете писать соответствующее название сложности буквами. Вы должны указать номер !!!

        WARNING: Уровень, создаваемый вами с помощью новой сложности, это полностью новый уровень.
        Прогресс из любого уровня не переносится в новый.
